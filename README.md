# Fishy RL

![Documentation](https://github.com/Oafish1/FishyRL/actions/workflows/build-docs.yml/badge.svg)

**Fishy RL** is a distributed reinforcement learning framework for model-based algoriths and Dreamer-V3. It is designed to be *flexible* and *user-friendly*, allowing researchers and practitioners to easily interchange components and environments. Please check out the [documentation](https://oafish1.github.io/FishyRL/) to get started.
